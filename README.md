# 🚀 Syncro

Syncro is a real-time collaborative workspace platform that enables teams to organize tasks, assign roles, and track progress using a live Kanban board system.

It combines structured team management with instant synchronization, eliminating communication gaps in collaborative projects.

---

## 🌍 Overview

Syncro allows users to:

- Create or join workspaces  
- Assign structured roles within teams  
- Manage tasks visually using a Kanban board  
- Receive real-time updates and notifications  
- Track activity history within a workspace  

The platform is designed to reduce workflow friction and improve team accountability.
